Q&A

창업에듀에서 자주하시는 질문과 답변을 모았습니다.

전체
  • 시스템문의
  • 과정등록문의
  • 학습문의

  • 카테고리명

    질문

  • [학습문의]

    수료기준이 궁금합니다.

  • [개별강좌]

    강의를 재생 후 끝까지 시청하면 수료됩니다.


    [패키지과정]

    수강신청 후 구성 강좌 리스트에 있는 전체 강의 시청 및 퀴즈풀기를 진행해야 수료됩니다.

  • [시스템문의]

    동시 접속으로 인해 수료처리가 되지 않습니다.

  • 창업에듀는 동시 수강이 불가함에 따라 아래와 같은 환경에서

    동시진행으로 인한 진도율 미반영 팝업이 발생할 수 있는 점 학습에 참고 부탁드립니다.


    1. PC 와 모바일로 동시 로그인하여 학습하는 경우

    2. 다른 브라우저로 동시 로그인하여 학습하는 경우


    위의 사항에도 해당하지 않는 경우 쿠키 삭제 후 학습을 진행해주시기 바랍니다.


    * 쿠키삭제 진행

        · 크롬(Chrome) :

          설정 > 개인정보 및 보안 > 인터넷 사용 기록 삭제 > 삭제 > 전체 창 종료 > 창업에듀 접속

        · 엣지(Microsoft Edge) :

          설정 > 개인정보 및 보안 > 인터넷 사용 기록 삭제 > 삭제 > 전체 창 종료 > 창업에듀 접속

        · 웨일(Naver whale)

          설정 > 개인정보 및 보안 > 인터넷 사용 기록 삭제 > 삭제 > 전체 창 종료 > 창업에듀 접속


    동일한 오류가 지속적으로 발생할 경우

    학습지원센터 (070-5066-0407) 혹은 1:1 문의게시판에 남겨주시면 답변드리겠습니다.

    수료증 발급은 어떻게 하나요?

  • 1. 개별강좌

    [마이페이지] → [나의 강의실] → 학습완료 개별강좌 → '수강확인증' 버튼 클릭

    * 개별강좌 통합 수료증이 필요한 경우 전체 선택 후 오른쪽 상단 '통합 수강확인증 출력' 클릭


    2. 패키지과정

    [마이페이지] → [나의 강의실] → 학습완료 패키지 과정 → '수료증' 버튼 클릭

    * 패키지과정은 개별 수료증 발급만 가능합니다.
